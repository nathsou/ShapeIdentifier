# ShapeIdentifier
Detect some shapes in an image thanks to opencv.

![Example](http://i.imgur.com/9q7YBoH.png)

![Example](http://i.imgur.com/9EfnzXz.png)

#Recognize simple polygons:

  - Green -> Polygon
  - Blue  -> Regular polygon
  - Red   -> Ellipse or Circle
